## 🎬 动画效果

### 淡入动画
```typescript
Animated.timing(fadeAnim, {
  toValue: 1,
  duration: 300-400ms,
  useNativeDriver: true,
})
```

### 弹簧动画
```typescript
Animated.spring(scaleAnim, {
  toValue: 1,
  friction: 8,
  tension: 40,
  useNativeDriver: true,
})
```

**应用场景：**
